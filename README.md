# 🎮 Neon Dodge

**Neon Dodge**는 Python과 pygame을 사용해 제작한 캐주얼 생존 게임입니다.  
플레이어는 네온 블루 색상의 블록을 조작하여, 화면 위에서 떨어지는 네온 핑크 장애물들을 피하며 최대한 오래 생존하는 것이 목표입니다.

게임은 단순한 조작 방식으로 시작하지만, 플레이 시간이 길어질수록 난이도가 점진적으로 상승하도록 설계되어 몰입감 있는 플레이 경험을 제공합니다.

## 🎯 게임 목표 및 규칙

- 플레이어는 좌우 방향키를 이용해 블록을 이동할 수 있습니다.
- 화면 상단에서 무작위로 생성되는 장애물들을 피하며 생존 시간을 늘려야 합니다.
- 시간이 지날수록 장애물의 생성 빈도와 속도가 증가하여 난이도가 점점 높아집니다.
- 플레이어에게는 제한된 **목숨(Lives)** 이 주어지며, 장애물과 충돌할 경우 목숨이 감소합니다.
- 모든 목숨을 소진하면 게임이 종료됩니다.

---

## ⚙️ 주요 기능 설명

### 1️⃣ 플레이 시간 기반 난이도 상승
플레이어의 생존 시간(점수)에 따라 게임 난이도가 자동으로 증가합니다.

- 장애물의 낙하 속도 증가
- 장애물 생성 간격 감소

이를 통해 초반에는 누구나 쉽게 적응할 수 있고, 후반으로 갈수록 집중력과 반사 신경이 요구되는 구조를 만들었습니다.

---

### 2️⃣ 랜덤 장애물 시스템
각 장애물은 생성 시 다음 요소들이 무작위로 결정됩니다.

- 크기 (작을수록 더 빠르게 이동)
- 낙하 속도
- 일부 장애물의 좌우 이동 여부

이로 인해 매 판마다 다른 패턴이 형성되며, 예측하기 어려운 상황 속에서 플레이어의 판단력이 중요해집니다.

---

### 3️⃣ 목숨(Lives) 시스템
플레이어는 처음에 정해진 개수의 목숨을 가지고 시작합니다.

- 한 번의 실수로 즉시 게임이 종료되지 않도록 설계
- 긴장감을 유지하면서도 공정한 게임 플레이 제공
- 반복 플레이와 도전 욕구를 유도

---

### 4️⃣ 점수 및 최고 점수 저장
- 현재 플레이 점수는 실시간으로 화면에 표시됩니다.
- 최고 점수(High Score)는 파일로 저장되어 게임을 다시 실행해도 유지됩니다.
- 게임 오버 화면에서 **이번 판 점수와 최고 점수**를 함께 확인할 수 있습니다.

---

### 5️⃣ 재시작 및 종료 기능
게임 오버 후 사용자는 다음 중 하나를 선택할 수 있습니다.

- `R` 키: 즉시 게임 재시작
- `Q` 키: 게임 종료

이를 통해 프로그램을 다시 실행하지 않고도 연속적인 플레이가 가능합니다.

---

## 🎮 조작 방법

| 키 입력 | 기능 |
|------|------|
| ← / → | 플레이어 좌우 이동 |
| R | 게임 오버 후 재시작 |
| Q | 게임 종료 |

## 🗂️ 프로젝트 구조 설명

본 프로젝트는 기능별로 파일을 분리한 패키지 구조로 설계되었습니다.

- `main.py`  
  프로그램 실행 진입점

- `settings.py`  
  화면 크기, 색상, 속도, 난이도 관련 상수 관리

- `player.py`  
  플레이어 객체 및 이동 로직 관리

- `obstacle.py`  
  장애물 생성, 랜덤 속성, 이동 패턴 구현

- `engine.py`  
  게임 루프, 충돌 판정, 점수 및 상태 관리

- `score.py`  
  최고 점수 파일 저장 및 불러오기

이와 같은 구조를 통해 코드의 가독성과 유지보수성을 높였습니다.
